# Data Model: Physical AI & Humanoid Robotics Textbook Platform

**Feature**: Physical AI & Humanoid Robotics Textbook Platform
**Branch**: `001-textbook-platform`
**Date**: 2025-12-01

## Entities

### Student
*   **Description**: Represents a user of the textbook and chatbot. Can be unauthenticated or authenticated.
*   **Attributes**:
    *   `id`: Unique identifier (UUID).
    *   `name`: Student's name (string, optional).
    *   `email`: Student's email address (string, optional, mandatory for authenticated).
    *   `software_background`: Description of student's software expertise (string, for authenticated, optional).
    *   `hardware_background`: Description of student's hardware/robotics expertise (string, for authenticated, optional).
    *   `language_preference`: Preferred language for content (string, e.g., 'en', 'ur', for authenticated, optional).
*   **Relationships**:
    *   Can have multiple `ChatbotQuery` and `ChatbotResponse` interactions.
    *   Can interact with `Textbook Content` for personalization.

### Textbook Content
*   **Description**: Represents a unit of educational material within the textbook (e.g., a chapter, section, or page).
*   **Attributes**:
    *   `id`: Unique identifier (UUID or Docusaurus-generated slug).
    *   `title`: Title of the content unit (string).
    *   `body`: Full text content of the unit (string, Markdown format).
    *   `language`: Language of the content (string, e.g., 'en', 'ur').
    *   `metadata`: JSONB, potentially including tags, keywords, difficulty level, or other attributes for personalization.
*   **Relationships**:
    *   Can be referenced by `ChatbotResponse`.
    *   Can be personalized for a `Student`.

### Chatbot Query
*   **Description**: Represents a question posed by a student to the RAG chatbot.
*   **Attributes**:
    *   `id`: Unique identifier (UUID).
    *   `student_id`: ID of the `Student` making the query (UUID, optional for unauthenticated).
    *   `text`: The actual question text (string).
    *   `context_snippet`: User-selected text snippet providing context for the query (string, optional).
    *   `timestamp`: Time of the query (datetime).
*   **Relationships**:
    *   Belongs to a `Student`.
    *   Triggers a `ChatbotResponse`.

### Chatbot Response
*   **Description**: Represents the answer generated by the RAG chatbot in response to a `Chatbot Query`.
*   **Attributes**:
    *   `id`: Unique identifier (UUID).
    *   `query_id`: ID of the `Chatbot Query` it responds to (UUID).
    *   `response_text`: The generated answer text (string).
    *   `source_references`: Array of references to `Textbook Content` IDs or specific sections from which the answer was derived (array of strings/UUIDs).
    *   `timestamp`: Time of the response (datetime).
*   **Relationships**:
    *   Responds to a `Chatbot Query`.
    *   References `Textbook Content`.
